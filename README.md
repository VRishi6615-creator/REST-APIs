REST CRUD Application

A simple RESTful CRUD application built using Node.js, Express.js, and EJS. This project allows users to create, view, edit, and delete posts, similar to a basic Quora-style posting platform.

Features
Create new posts
View all posts
View individual posts
Edit existing posts
Delete posts
RESTful routing
Dynamic pages using EJS
Technologies Used
Node.js
Express.js
EJS
Method Override
UUID
HTML
CSS
Installation
npm install
node index.js

Open your browser and visit:

http://localhost:5000/posts
